# Hardware:
- MCU: Teensy 4.0
- Sensor: GY-521 MPU-6050 (Orientation sensor) & GY-BMP280 (Barometer)
- Battery: 2S 1100mAh
- ESC: Skywalker 15A V2 ESC
- Motor: LANNRC 1404 4600kV Plus
